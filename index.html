How It Works
4-Step Detection Pipeline
01
🔍
Discover
Content script crawls the DOM, finds every image, extracts src or converts canvas to base64.
02
🛡️
Detect
Reality Defender API analyses pixel-level patterns — GAN artifacts, frequency anomalies, inconsistent noise.
03
🤖
Explain
Gemini translates the raw verdict into plain English — what looks wrong and why it matters.
04
📰
Verify
Google Fact Check cross-references related claims, surfacing existing debunks and context.
Architecture
Built on Chrome MV3
🧩
Content Script
Injected into every page. Discovers images, renders verdict overlays, handles user interactions and per-image rescans.
DOM traversal Canvas API CSS overlays
⚙️
Background Service Worker
Orchestrates the full scan pipeline. Checks cache first, calls APIs in sequence, stores results, handles fallback.
MV3 worker Fetch API Message passing
⚡
Smart Cache Layer
Every scan result stored in chrome.storage.local keyed by image hash. Rescans return instantly without hitting APIs.
chrome.storage Request dedup ~50ms rescans
🛡️
Fallback Mode
When any API is unreachable, PixelProof generates a simulated verdict locally. The demo never breaks. Keys stay local only.
Local simulation Diagnostic logs Zero key exposure
Features
Everything, In the Browser
🖼️
Live Image Scanning
Scans every image on any page. Works on news sites, social feeds, anywhere images load in Chrome.
🤖
AI Explanations
Gemini explains why an image looks suspicious — not just a score, but understandable reasoning.
📰
Fact-Check Context
Cross-references image claims against Google's Fact Check database. Surfaces existing debunks instantly.
⚡
Instant Cache
Scan an image once, get results forever. Results persist in chrome.storage.local. Rescans are ~50ms.
🔑
Local Key Storage
API keys never leave your device. Stored in chrome.storage.local, never bundled in the extension.
🛡️
Graceful Fallback
Network down? Key missing? PixelProof simulates locally. Presentations never break mid-demo.
